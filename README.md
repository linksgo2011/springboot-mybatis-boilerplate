# springboot-mybatis-boilerplate

A boilerplate Template project for spring boot with mybatis.

## features

- Flyway DB migration
- Base CRUD with MyBatis  
- Pagination 
- Common mapper
- One to Many relationship >> current
- Auth system
- Base CRUD Api test
- Criteria Query
- Authentication and Authorization
- Arch unit
- More
